El login permite identificar a un usuario antes de darle acceso a una aplicación. Gracias a la autenticación, solo las personas autorizadas pueden utilizar determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, claves de API ni otros datos sensibles. Proteger esta información ayuda a mantener la privacidad y la seguridad de la plataforma.
