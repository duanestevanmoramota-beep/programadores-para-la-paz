Durante este curso aprendí que la tecnología puede ayudar a fortalecer la participación ciudadana mediante herramientas que permiten registrar información, organizar datos y facilitar la comunicación entre las personas. También comprendí la importancia de utilizar Git y GitHub para controlar versiones de los proyectos, así como Node.js para crear aplicaciones. Además, aprendí sobre seguridad, automatización y protección de datos. Estos conocimientos permiten desarrollar plataformas más organizadas, seguras y útiles para la comunidad.
