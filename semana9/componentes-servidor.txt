Sistema operativo:
Es el software principal que administra los recursos del computador y permite ejecutar programas. Gestiona memoria, archivos y procesos.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo donde se encuentra el código del servidor. Define las rutas y la forma en que responde a las solicitudes.

Puerto:
Es un canal de comunicación que utiliza el servidor para recibir peticiones. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad del servidor. Ayudan a detectar errores y monitorear el funcionamiento del sistema.
