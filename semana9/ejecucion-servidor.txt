Al ejecutar el comando node server.js se inicia el servidor Node.js utilizando Express.

En la terminal aparece el mensaje:

"Servidor ejecutándose en puerto 3000"

Cuando se visita la ruta http://localhost:3000/estado, el servidor responde con un objeto JSON que indica que el servicio está funcionando correctamente.

Respuesta obtenida:

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
