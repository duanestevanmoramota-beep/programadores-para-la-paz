Repositorio: Un repositorio es el espacio donde se almacenan los archivos de un proyecto y todos los cambios realizados. Permite organizar el trabajo y mantener un historial de versiones para facilitar la colaboración y recuperación de información.

Commit: Un commit es un registro de los cambios realizados en un proyecto en un momento específico. Cada commit guarda una versión del trabajo y permite identificar qué modificaciones se hicieron y cuándo.

Push: El comando push se utiliza para enviar los cambios del repositorio local hacia un repositorio remoto como GitHub. Esto permite compartir el proyecto y mantenerlo actualizado en línea.
